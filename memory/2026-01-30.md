### Peter's AI Development Workflow (Aug 2025 blog post)

**Setup:**
- Ghostty terminal + Claude Code as main driver
- VS Code on the side for code review
- Dell UltraSharp U4025QW (3840x1620) - fits 4 Claude instances + Chrome
- Cursor/GPT-5 for reviews

**Multi-Agent Strategy:**
- Small tasks: 1-2 agents
- Cleanup/tests/UI work: ~4 agents
- All work on main branch (tried worktrees, slowed him down)
- Pick areas of work carefully to avoid cross-pollination

**Key Principles:**
1. **Context is precious** - Tests written IN SAME CONTEXT find more issues
2. **CLI-first integrations** - "Pick services with CLIs: vercel, psql, gh, axiom. One line in CLAUDE.md is enough"
3. **Less is more** - Removed his last MCP because Claude would spin up Playwright when reading code was faster
4. **Plan mode + iteration** - Bigger tasks written to file, reviewed by GPT-5
5. **Refactor constantly** - "Claude often makes a mess but it's equally great in refactoring"

5. **Avoid MCP bloat** - Keep tools simple, let agents read code directly when possible
